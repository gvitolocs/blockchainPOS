1) Message framing and serialization
We send each message as JSON. Before the JSON, we send 4 bytes with the
message length (big-endian). The receiver reads the 4 bytes, then reads
exactly that many bytes and decodes the JSON.

2) How to run handin0
Open a terminal and run:
cd dissy_da1_1_hand-ins-handin0/peer
go run .

This starts three peers on different ports and shows Ping/Pong logs.

3) How to run tests
Open a terminal and run:
cd dissy_da1_1_hand-ins-handin0/peer
go test
